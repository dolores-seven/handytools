# 科幻词卡生成器

一个赛博朋克风格的引言/金句卡片生成工具。

## 功能

- 赛博朋克暗色主题
- 双语（中英）引言展示
- 自定义背景色、边框色、文字色
- 可调边框宽度
- 多种英文字体可选
- 导出 PNG / SVG
- JSON 导入/导出

## 使用方式

### 方式一：直接打开（推荐）

双击 `generator.html` 在浏览器中打开，手动编辑内容后导出。

### 方式二：通过 Claude Code Skill

在 Claude Code 中输入：
```
/科幻词卡
```
然后提供引言内容，Claude 会自动生成 HTML 文件。

## 默认样式

- 背景：纯黑 `#000000`
- 边框/文字：荧光绿 `#00FF00`
- 边框宽度：4px
- 中文部分使用汇文明朝体，英文使用 Courier New
