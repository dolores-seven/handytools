# 英语单词卡片生成器

一个简洁的英语单词学习卡片制作工具。

## 功能

- 输入单词、音标、释义、例句
- 4 种卡片主题：经典白底、深蓝金字、复古棕黄、薄荷炭黑
- 导出高清 PNG 图片
- 卡片 3:4.5 比例，适合手机壁纸或打印

## 使用方式

### 方式一：直接打开（推荐）

双击 `generator.html` 在浏览器中打开。

操作步骤：
1. 填写单词信息（单词为必填，其余可选）
2. 选择喜欢的卡片主题
3. 点击"生成卡片"预览效果
4. 点击"下载卡片"保存为 PNG

### 方式二：通过 Claude Code Skill

在 Claude Code 中输入：
```
/英语单词卡
```
然后提供单词列表，Claude 会自动生成 HTML 文件。
