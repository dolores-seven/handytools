# 优雅卡片生成器

一个精美的双语金句/引言卡片生成工具。

## 功能

- 双语（中英）引言卡片
- 多种中英文字体可选（汇文明朝体、思源宋体、马善政行草等）
- 自定义配色（文字颜色、背景颜色）
- 4 种预设配色模板（深蓝金、黑米色、深靛蓝、皇家紫）
- 导出 PNG / SVG
- 导出/导入 JSON 配置

## 使用方式

### 方式一：直接打开（推荐）

双击 `generator.html` 在浏览器中打开，手动编辑内容后导出。

### 方式二：通过 Claude Code Skill

在 Claude Code 中输入：
```
/优雅卡片
```
然后提供你要做成卡片的内容（引言、署名等），Claude 会自动生成 HTML 文件。

## 字体说明

默认使用**汇文明朝体**（Huiwen-mincho），通过 CDN 自动加载。
- 首次加载可能需要几秒钟（字体文件较大）
- 加载期间会显示回退字体（思源宋体）
- 如需本地安装以获得最佳体验，可从 [GitHub](https://github.com/bosswnx/huiwenmincho-improved) 下载安装
