# Handy Tools

一些好用的小工具，直接浏览器打开即用，无需安装。

## 工具列表

### 优雅卡片生成器 `elegant-card/`

生成精美的双语（中英）金句/引言卡片，支持自定义字体、配色，导出 PNG/SVG。

- 多种中英文字体（汇文明朝体、思源宋体、马善政行草等）
- 4 种预设配色模板
- 导出 PNG / SVG / JSON

### 科幻词卡生成器 `scifi-card/`

生成赛博朋克风格的引言/金句卡片。

- 赛博朋克暗色主题
- 自定义背景色、边框色、文字色、边框宽度
- 导出 PNG / SVG

### 英语单词卡片生成器 `vocab-card/`

生成英语单词学习卡片。

- 支持音标、释义、例句
- 4 种卡片主题
- 导出高清 PNG

## 使用方式

所有工具都是单个 HTML 文件，双击即可在浏览器中打开使用。

无需安装任何依赖，无需服务器，无需注册账号。

## 技术栈

- 纯前端 HTML/CSS/JavaScript
- [Tailwind CSS](https://tailwindcss.com/)（CDN）
- [html2canvas](https://html2canvas.hertzen.com/)（导出 PNG）
- [Google Fonts](https://fonts.google.com/)（在线字体）
